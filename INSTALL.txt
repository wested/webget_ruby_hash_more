
= WebGet.com Ruby Gem Install


First-time users: add our gem certificate and source.
When you do this once, it works for all our gems.

  sudo wget http://webget.com/webget.pem
  sudo gem cert --add webget.pem   
  sudo gem sources --add http://webget.com

Install the gem with advanced options.

  sudo gem install webget_ruby_hash_more --test --trust-policy HighSecurity
